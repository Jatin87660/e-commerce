# E-Commerce Website
## Description
This is a e-commerce website for managing users, items, carts, and orders. It allows users to register, login, create items, add items to carts, and convert carts to orders.

## API Endpoints

### Users
- `POST /users`        : Creates a new user
- `GET /users`         : List all users
- `POST /users/login`  : Login with username and password

### Items
- `POST /items`        : Creates an item
- `GET /items`         : List all items

### Carts
- `POST /carts`        : Create a cart and add items
- `GET /carts`         : List all carts

### Orders
- `POST /orders`       : Convert a cart to an order using cart ID
- `GET /orders`        : List all orders
